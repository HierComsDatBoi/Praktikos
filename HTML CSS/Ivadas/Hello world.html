Tag sarasas

  • html - talpina visa sveitaines koda
  • head - informacija apie sveitaine
  • body - svetaines atvaizduojamas turinys
  • meta - svetaines meta informacija
  • link - prijungti papildomus failus
  • script - skirtas prijungti JS ir TS failus
  • title - puslapio tabo pavadinimas
  • p - paragrafas teksto turinys
  • li - list item, elementas su turiniu
  • ul - unordered list
  • ol - ordered list
  • h1-h6 - heading dydis
  • a - anchor nuoroda
  • img - paveiksliukas

Atributu sarasas

  • charset - kokia simboliu biblioteka puslapis naudoja
  • rel - relation nurodo link tago santyki
  • href - hyperlink reference, failo kelias arba nuorodos linkas
  • src - source failo internete arba tinklapio nuoroda
  • alt - alternative text, tekstas kuris yra jeigu paveiksliukas neveikia

Klaviaturos trumpiniai

  • win+shift+s - snipping tool
  • alt+arrow up|down - perkelia eilute i virsu|apacia
  • ctrl + shift + k - istrina eilute
  • ctrl enter nauja eilute atidaro kursoriui esant betkur kitoj eilutej
