Breakpoints

Breakpoint - ekrano luzio dydis, kuri naudojant keisis elementu isdestymas

  1. X-Small < 576px
  2. Small ≥ 576px
  3. Medium ≥ 768px
  4. Large ≥ 992px
  5. Extra large ≥ 1200px
  6. Extra extra large ≥ 1400px

Query aprasymas - @media (min-width: px){...css...}
pradzioj css apsiraso pastovus stiliai, veliau apacioje aprasomi dydziai kurie pasikeis.
nuo maziausio iki didziausio

@media nesuteikia papildomo prioritete, todel reikia teisingo eiliskumo.
media stiliai nepradingsta pasiekus nauja media query

Mobile first

dizaino principas toks, kad pradzioje puslapis pritaikomas telefonams, tada judama prie didesniu dydziu
Taip daroma, kad didesnis kiekis naudojasi telefonu, maziau vietos atvaizduoti informacija, tai leidzia atsirinkti svarbesne info

  2. Atsirenkami svarbesni elemenentai, kai maziau vietos
