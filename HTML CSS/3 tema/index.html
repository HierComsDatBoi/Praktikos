Teviniai vaikiniai

Elementas turi tik viena tevini elementa

Paveldimumas

Stiliai gali buti paveldimi is tevinio elemento i vaikini elementa

Stiliu prioritetai

  1. !important
  2. Inline Style
  3. Specifikacija (selector specificity)
    1. ID (1, 0, 0)
    2. Class (0, 1, 0)
    3. Element (0, 0, 1)
  4. Eiliskumas - toliau faile aprasytas
  5. User agent stylesheet - svetaines numatytieji stiliai
  6. Inheritance - Paveldimumas
