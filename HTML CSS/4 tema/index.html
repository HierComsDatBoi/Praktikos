Padding

Tarpas tarp turinio ir remelio
Gali buti nustatomas

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
  • padding: t r b l - clockwise

Border

Juosta atskirianti elemento padding nuo margin

Uz borderio baigiasi background dalis

Galima suteikti spalvos, storio, linijos tipo, kapmpu apvalinimo stilius skirtingus stilius galima suteikti skirtingoms krastinems

  • border: width style color;
  • border-width: px;
  • border-style: solid/dashed/dotted/double;
  • border-color: red;
  • border-[krastine]-[color,style,width]: ;
  • border-radius: 5px;
  • border-radius: 30px 5px 20px;

Margin

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • margin: t r b l - clockwise
  • margin: all
  • margin: 0 auto;

Tarpas tar elemento remelio ir aplinkiniu elementu
Marginai gali persidengti jeigu vienas po kitu einantys elementai yra blokiniai

Margin: 0 auto; naudojant su width atributu iscentruoja blokini elementa is krastu

Margin is virsaus ir apacios neveikia inline elementams

Box Sizing

box-sizing: border-box; sudtalpina padding margin i content

Selectoriai

  • selector1, selector2, ...- keletos selektoriu parinkimas
    • tagName.class
    • tagName.class.class
    • tagName.class#id
    • tagName#id
    • .class.class
    • #id.class
  • selector1 selector2 - selector1 viduje nesvarbu kaip giliai esantis selector2
  • selector1 > selector2 .... - selector1 viduje is karto einantis vaikinis selector2
  • selector1 + selector2 - is karto einantis selector2
  • selector1 ~ selector2 - uz selector 1 einantys (daugiskaita) selector2
