Metodai

Metodas - egzistuojanti funkcija kuri priklauso kazkokiam specifiniam elementui ar kintamajam. savybe/property - nekintanti reiksme

Metodai gali buti prototipinai (instance) arba statiniai (static).
Prototipinio metodo atveju kreipiames i metoda ar savybe per kintamaji.
Statinio metodo atveju kreipiames i metoda ar savybe per globaluji kintamaji.

tipas.prototype.metodas/savybe - kintamasis.metodas()/savybe
Tipas.metodas/savybe - Tipas.metodas()/savybe

Savybes

  • .Number.EPSILON - grazina epsilon
  • .Number.MAX_SAFE_INTEGER
  • .Number.MAX_VALUE
  • .Number.positive_infinite
  • .Number.min_safe_integer
  • .Number.max_value
  • .Number.negative_infinite
  • .Number.NaN

statiniai metodai

  • .isFinite(sk) - ar skaicius begalinis
  • .isInteger(sk) - ar skaicius sveikasis
  • .isSafeInteger(sk)
  • .isNaN(kazkas)
  • .parseInt(kazkas) - pavercia skaiciu arba stringa i sveikaji skaiciu
  • .parseFloat(kazkas) - pavercia skaiciu arba stringa i realuji skaiciu
  • .parseFloat(kazkas)

instance prototipiniai metodai

  • .prototype.toPrecision(sk)
  • .prototype.toFixed(sk) - sk kiek skaiciu uz kablelio suapvalintai palieka - grazina string
  • Number.parseFloat(sk.toFixed(4)) pavercia string i number
  • +sk.toFixed(4) pavercia string i number

String metodai

  • .length - grazina stringo simboliu kieki
  • .prototype.at(nr) - nurodo string'o nr pozicijoje esantį simbolį (galima skaičiuoti nuo galo)
  • .prototype.charAt(nr) - nurodo string'o nr pozicijoje esantį simbolį
  • .prototype.charCodeAt(nr) - grąžina nurodytos raidės UTF-16 bibliotekoje kodą
  • .prototype.concat(string1, string2, string3...) - apjungia du arba daugiau string'us į vieną
  • .prototype.startsWith(text) - ar string prasideda su text
  • .prototype.endsWith(text) - ar string pasibaigia su text
  • .prototype.includes(text) - ar string'e yra text'as
  • .prototype.indexOf(text) - grąžina indeksą pirmojo rasto nurodyto text'o
  • .prototype.lastIndexOf(text) - grąžina indeksą paskutinio rasto nurodyto text'o
  • .prototype.localeCompare(text) - palygina string'ą su text'u
  • .prototype.match(regex) - ar atitinka bent vieną regex
  • .prototype.matchAll(regex) - ar atitinka visus regex
  • .prototype.padStart(ilgis, text) - prideda text pradžioje kol pasiekiamas ilgis
  • .prototype.padEnd(ilgis, text) - prideda text gale kol pasiekiamas ilgis
  • .prototype.repeat(kiek) - kiek kartų atkartoti string'ą
  • .prototype.replace('ka', 'kuom') - pakeičia pirmąjį rastą 'ka" į 'kuom'
  • .prototype.replaceAll('ka', 'kuom') - pakeičia visus rastus 'ka" į 'kuom'
  • .prototype.search(regex)
  • .prototype.slice(nuo, iki) - atkerpa iš string dalį nuo(imtinai) iki iki(neimtinai)
  • .prototype.substring(nuo, iki) - atkerpa iš string dalį nuo(imtinai) iki iki(neimtinai)
  • .prototype.split(skirtukas) - sukarpo string'ą į gabaliukus (pagal skirtukas) ir grąžina masyvą
  • .prototype.toLowerCase() - paverčia visą string į mažąsias raides
  • .prototype.toUpperCase() - paverčia visą string į didžiąsias raides
  • .prototype.trimStart() - panaikina tarpus stringo pradžioje
  • .prototype.trimEnd() - panaikina tarpus string'o pabaigoje
  • .prototype.trim() - panaikina tarpus ir pradžioje ir pabaigoje
