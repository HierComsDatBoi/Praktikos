Galima html pildyti naudojant innerHTML - rasant html koda js kaip string ir juo papildyti html elementa. tuomet tas stringas html bus skaitomas kaip html kodas ir atvaizdojamas kaip html kodas

Geresnis budas papildyti html su js - kurti nodus (html elementus) juos prilyginnti kintamiesiems ir tuomet tuos nodus appedinti i kitus html elementus.
node yra kuriami, odifikuojami, iterpiami i html naudojant node metodus

Node metodai

  • document.createElement('tagName');
  • document.createTextNode('tekstas') - galima nenaudot
  • kam.append(ka1, ka2, ka3....) - i koki node (kam) gale idesim (ka) nodus ar teksta
  • kam.appendChild(ka) - i kokio node (kam) gala idesime (ka)
  • kam.prepend(ka1, ka2, ka3....) - prideda i pradzia
  • kam.setAttribute('name', 'value')
  • kam.classList
    • add(class1, class2, class3....)
    • remove(class1, class2, class3....)
    • replace(classNuemi, classPridedi)
    • toggle(class) - jei klase yra ja isima, jei nera ja ideda
