komponentai

Komponentas - tai HTML objektas sukurtas JS

Komponentai dazniausiai kuriame frameworku, kaip react/vue/angular/swift pagalba, bet galima sukurti ir JS kaip atskiras klases ar funkcijas

Moduliai

moduliai - skirtingi/atskiri failai su savo kakokiu kodu kurie gali buti panaudojami kitu failu.

  • moduliuose gali buti ivairus kodas. Kintamieji, funkcijos, klases....
  • klases turi buti atskiri failai. Viena klase - vienas failas
  • HTML prijungtas JS kuris kazka importuoja turi tureti atributa type="module" ir jam tada nereikia defer
  • HTML failas prie kurio yra prijungtas modulis privalo buti paleistas kaip serveris, kitu atveju modulis nebus prijungtas

export ir import

Failuosde esantis kodas gali buti iskeliamas ir galima kazkoki koda is kito failo isikelti

export

  • toje pacioje eiluteje (negalima export default ()=>{} (su arrow func) )
  • kodo apacioje isvardinti visus export arba viena export default atskirose eilutese.

import

  • import defaultRandomName
  • import {simple, simple as different}
  • import defaultRandomName, {simple, simple as different}
